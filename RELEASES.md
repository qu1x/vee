  * Use Unicode *combining diacritical marks* instead of ~/L/R.
  * Define 6D/7D PGA blades.

  * Distinguish motors.
  * Add more examples.
  * Add polarity operator.
  * Fix bugs, typos, and improve documentation.
  * Explore 4D/5D PGA.
